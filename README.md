# 🎂 RK Bakery Website

A complete, professional bakery website built with HTML, CSS & JavaScript.

## Files

| File | Description |
|------|-------------|
| `rk-bakery.html` | **Single-file version** (all-in-one, just open in browser) |
| `index.html` | HTML structure |
| `style.css` | All styles (1,070 lines) |
| `script.js` | All JavaScript (455 lines) |

## Features
- ✅ Responsive mobile-friendly layout
- ✅ Sticky navbar with active link tracking
- ✅ Hero section with animated cake illustration
- ✅ Full cake menu (Butter Cream + Fresh Cream + Custom)
- ✅ Sweets menu (10 items)
- ✅ Live search filtering across cakes & sweets
- ✅ WhatsApp order buttons on every product
- ✅ Floating WhatsApp button
- ✅ Customer reviews carousel with auto-scroll
- ✅ Gallery section
- ✅ Contact form (validates + opens WhatsApp)
- ✅ Footer with social media links
- ✅ Scroll-reveal animations
- ✅ Back-to-top button

## Quick Start
Open `rk-bakery.html` in any browser — no server needed!

## Customisation
- **Phone number**: Search `919876543210` in all files and replace with your number
- **Address**: Update in `index.html` contact section
- **Prices**: Edit the data arrays at the top of `script.js`
- **Colors**: Edit CSS variables in `:root` block in `style.css`
